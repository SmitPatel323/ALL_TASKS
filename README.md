# Web Projects - Task 1 & Task 2

A collection of responsive web projects built with modern web technologies.

## 📋 Projects Overview

### Task 1 — Responsive Webpage
A fully responsive single-page website featuring:
- **Header** with logo and navigation (hamburger menu on mobile)
- **Hero Section** with call-to-action button and placeholder image
- **About Section** with company description
- **Services Section** showcasing key services
- **Footer** with contact info and social media links

**Tech Stack:** HTML5, Tailwind CSS, Vanilla JavaScript

### Task 2 — Product Listing with Sorting
An interactive product management page with:
- Product cards displaying items with pricing and categories
- Filter to show only in-stock products
- Dynamic sorting by price (ascending/descending)
- Responsive grid layout
- Hover effects and smooth transitions

**Tech Stack:** HTML5, Tailwind CSS, Vanilla JavaScript

---

## 🛠 Tech Stack

- **HTML5** — Semantic markup
- **Tailwind CSS** — Utility-first CSS framework for styling
- **Vanilla JavaScript** — No dependencies, pure JS for interactivity
- **Placehold.co** — Free placeholder images

## 🚀 How to Run

### Option 1: Direct Browser Opening
1. Navigate to the project folder
2. Open `TASK_1/index.html` or `TASK_2/index.html` in your browser
3. No build process needed!

### Option 2: Using Live Server (VS Code)
1. Install the "Live Server" extension in VS Code
2. Right-click on `index.html` and select "Open with Live Server"
3. Browser will auto-reload on file changes

### Option 3: Using Python (Simple Server)
```bash
cd EXAM_TASKS/TASK_1
python -m http.server 8000
# Visit http://localhost:8000
```
